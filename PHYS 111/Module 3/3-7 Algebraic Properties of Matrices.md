Note: $A$, $B$, and $C$ are matrices, $c$ and $d$ are scalar
1. $A + B$ = $B+A$
2. $(A+B) + C = A + (B + C)$
3. $c(A+B) = cA + cB$
4. $(c+d)A = cA + dA$
5. $c(dA) = (cd)A$
6. 
	a. $A(B+E) = AB + AE$
	b. $(A+B)E = AE + BE$
7. $(AB)E = A(BE)$
8. $(A+B)^T = A^T + B^T$
9. $(cA)^T = cA^T$
10. $(AB)^T = B^TA^T$
11. $A(cB) = c(AB) = cAB$
